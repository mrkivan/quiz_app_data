WealthFlow — Privacy Policy (Version 2.0)

Effective date:

Summary

WealthFlow is a local-first personal finance app. Version 2.0 introduces limited analytics strictly for app stability and UI interaction improvement. These analytics do not include personal, financial, or identifiable user data. All your financial data remains stored locally on your device and encrypted.

App features

WealthFlow helps you manage your personal finances with features including:

  • Assets, investments, and liabilities tracking.
  • Secure 6-digit passcode protection for the app.
  • Local currency formatting and multi-language UI options.
  • Local, on-device only storage of all finance records and profile info.
  • Improved UI and optimized local database in version 2.0.

Data collection (updated in v2.0)

WealthFlow now collects minimal analytics for stability and performance improvements.

Specifically, the app collects:

  • Crash reports — used only to identify technical issues and app errors.
  • Anonymous screen visit events — to understand which screens are used most.
  • Anonymous button-click events — only button identifiers, not the data entered.

What is NOT collected:

  • No financial data (assets, investments, liabilities, notes).
  • No personal identity data.
  • No device contacts, SMS, files, or unrelated information.
  • No sensitive parameters or content from forms.

Analytics data is non-personal, non-sensitive, and cannot be used to identify you or reconstruct your financial data.

Local storage & encryption

All finance-related data is stored only on your device and protected with encryption.

Security details:

  • Data at rest is encrypted using platform-provided secure encryption (e.g., AES-based secure storage).
  • The 6-digit passcode, when enabled, protects access to the encrypted data store.
  • Keys are stored using secure platform facilities like Android Keystore or iOS Keychain.

No financial or personal data is ever uploaded as part of analytics or any other service.

No cloud services for financial data

WealthFlow does not upload your financial records to any cloud server. Crash logs and interaction analytics do not include any user-entered content.

  • Your finance data stays offline unless you manually export it.
  • No third-party tracking of financial or personal info.

Optional exports & sharing

If you choose to export or share finance data, those exported files are handled by your device's sharing system. WealthFlow will always request confirmation before generating export files.

Third-party libraries

WealthFlow uses third-party libraries for UI, encryption, database storage, crash reporting, and non-personal analytics. These libraries do not access your financial or personal data.

Children & minors

WealthFlow does not knowingly target children under 13. Because the app does not collect personal or financial data, it does not knowingly collect information from minors.

How to remove data

You can remove all app data by using the in-app deletion options or by uninstalling the app. Uninstalling removes all local financial data stored by WealthFlow.
